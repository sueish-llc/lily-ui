# Security Policy

## Supported versions

Only the latest published version of each `@lily-ui/*` package receives
security fixes.

## Reporting a vulnerability

Please **do not** open a public issue for security problems. Instead, report
them privately via
[GitHub private vulnerability reporting](https://github.com/sueish-llc/lily-ui/security/advisories/new).

Include what you can: affected package and version, reproduction steps, and the
impact you expect. You should hear back within a week. Once a fix is released,
the advisory is published and credit is given unless you ask otherwise.
