# AI 利用ガイド

常に読み込まれる正式な AI ガイドは、リポジトリのルートにあります。[`CLAUDE.md`](../CLAUDE.md) です。

ここには次の内容が書かれています。

- Lily UI が AI にやさしい構造にしてある理由
- コンポーネントの API 規約と、共有するプロップの語彙（`status` / `size` / `variant` / `block`）
- `.lily-*` のクラス名の付け方
- トークンの早見表（生の値ではなくセマンティックトークンを使うこと）
- テーマと密度の属性
- 生成コードのための do/don't ルール

これらのルールが表している見た目の世界観については、[design-language.md](design-language.md) を参照してください。
