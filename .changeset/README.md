# Changesets

This folder is managed by [Changesets](https://github.com/changesets/changesets).
Run `pnpm changeset` to record a change; CI uses these to version and publish the
packages. `@lily-ui/css` and `@lily-ui/react` are version-linked so they always
release together. See the
[docs](https://github.com/changesets/changesets/blob/main/docs/intro-to-using-changesets.md).
